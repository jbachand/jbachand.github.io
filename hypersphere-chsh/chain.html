Each ramp of the π-chain adds geometric resolution to the underlying trajectory. The shadow is a fixed-dimensional projection onto a chosen direction. As chain depth grows, that projection concentrates — the cloud of possible outputs collapses onto the equator perpendicular to the projection axis.

Math: a uniform point on Sn−1 has coordinate variance 1/n. As n → ∞, every coordinate → 0 in distribution. The point lives on the equator with respect to any axis. This is why the chain's 512-bit shadow stays fixed at any depth (Law 6, Law 31): the trajectory grows, but its projection doesn't.

// The tiny "chain" that produces what you see.
// Each particle is a uniform sample on the unit n-sphere S^(n−1),
// projected to first 3 coordinates and rendered on S² (the visible
// sphere).  As n grows, the projection concentrates on the equator
// perpendicular to the projection axis.

function gaussian() {
  // Box-Muller — standard normal sample
  const u = Math.random(), v = Math.random();
  return Math.sqrt(-2 * Math.log(u)) * Math.cos(2 * Math.PI * v);
}

function sampleLatitude(n) {
  // Returns u₁ where u is uniform on S^(n−1).
  //   exact:       u₁ = g₁ / √(g₁² + g₂² + … + gₙ²)
  //   asymptotic:  u₁ ≈ 𝒩(0, 1/n)  for n large
  //   limit:       u₁ = 0 at n = ∞ (the equator)
  if (n === Infinity) return 0;
  if (n <= 60) {
    let sum = 0;
    const g1 = gaussian();
    sum = g1 * g1;
    for (let i = 1; i < n; i++) {
      const g = gaussian();
      sum += g * g;
    }
    return g1 / Math.sqrt(sum);
  }
  return gaussian() / Math.sqrt(n);
}

function placeOnSphere(n) {
  // One particle: pick latitude from the n-sphere projection,
  // longitude uniform on [0, 2π).  Render at unit radius.
  const sinT = sampleLatitude(n);
  const cosT = Math.sqrt(1 - sinT * sinT);
  const phi  = Math.random() * 2 * Math.PI;
  return [cosT * Math.cos(phi), sinT, cosT * Math.sin(phi)];
}
