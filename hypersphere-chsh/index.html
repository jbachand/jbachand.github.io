The Hypersphere Does It

|S| = p√8 from Werner-state density-matrix algebra · Bachand 2026

Four self-contained demonstrations of the full CHSH spectrum from |S| = 0 to |S| = √8. Each runs in your browser in milliseconds. No installation, no dependencies — pure JavaScript reproducing the same arithmetic as the Python scripts on GitHub.

|S| = p · √8  where p = Werner singlet weight
Classical Bell bound |S| ≤ 2 p = 1/√2 ≈ 0.707
Linear march |S| = p√8 Werner γ-knob
Tsirelson bound |S| ≤ √8 p = 1 (pure singlet)

§3.1 — Shared-state classical counter

Two functions A and B share a global counter, ticking at incommensurate rates with a random phase offset (the hidden variable). CHSH outcomes are parity readouts at chosen sampling times. A fully classical LHV system reaching the Bell bound by Bell-saturation.

§3.2 — Deterministic zipper (complementary specialization)

  

    A is fast on odd N, slow on even; B is the reverse. Each function only
    ever completes its easy half; joint system advances at the fast rate.
    CHSH with disjoint Alice/Bob sampling grids gives exactly |S| = 2.0000
    — perfect classical saturation with zero noise.

§3.3 — Werner γ-knob (pure 4×4 density-matrix algebra)

  

    Construct the Werner state ρ_W(p) = p·|Ψ⁻⟩⟨Ψ⁻| + (1−p)·I/4, compute
    |S| via the Born-rule trace formula. As γ slides from 0 to 1, p slides
    from 1/√2 to 1, and |S| marches linearly from the Bell bound 2 to the
    Tsirelson bound √8. Drag the slider to see it live.

§3.4 — From-scratch hypersphere simulator

  

    Store |Ψ⁻⟩ as 4 complex amplitudes on S⁷. Build measurement projectors
    as 2×2 matrices, tensor them, compute Born expectations. The
    √8 is not assumed — it is computed from |Ψ⁻⟩'s geometric
    overlap with the optimal CHSH axes.
  

  

    
  

  

    
    

      
Singlet correlation

      
E(a, b) = −cos(a − b)

      
Each |aᵢ − bⱼ| = π/4

      
→ |Eᵢⱼ| = cos(π/4) = 1/√2

      
Closed-form values:

      
E₁₁ = −0.70711(target −1/√2)

      
E₁₂ = +0.70711(target +1/√2)

      
E₂₁ = −0.70711(target −1/√2)

      
E₂₂ = −0.70711(target −1/√2)

      

      
S = E₁₁ − E₁₂ + E₂₁ + E₂₂

      
|S| = 2.82843

      
target: √8 = 2.82843

  What this is and isn't.
  These demonstrations show the Werner-state CHSH theorem (Werner 1989;
  Horodecki et al. 1995) in transparent algebraic form. No new physics.
  The empirical content concerns a distinct non-classical boundary —
  Bell non-locality — which is distinct from Penrose's Gödel-based
  non-computability argument. Standard QM is Turing-computable
  (BQP ⊆ EXP); Bell non-locality therefore does not address algorithmic
  non-computability. See the
  paper
  for the careful scope.
